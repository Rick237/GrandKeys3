Ma3 Windows
This app is a keyboard shortcut trainer designed for MA3PC on Windows. It loads official MA3 shortcuts and challenges users to press the correct key combinations. The app tracks speed, accuracy, and mistakes during each run. Results can be saved to a leaderboard to compare performance with other users.

Ma3 Mac version
This version trains grandMA3 keyboard shortcuts on macOS, with support for Mac-specific keys like Option and Command. It presents random or sequential shortcuts and measures how quickly and accurately users respond. Performance statistics are shown after each run.

Custom version
This version allows users to train any set of shortcuts by loading their own XML file. The app reads the custom shortcut definitions and generates practice sessions automatically. It records speed, accuracy, and attempt statistics for each run. This makes it useful for learning custom workflows or other software shortcuts.
