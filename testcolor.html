Give me the color palette of grandma 3 The grandMA3 interface does not use a single fixed “brand palette” like a design system. Instead, its default UI theme uses a set of functional colors to indicate states (selected fixtures, tracked values, programmer data, etc.). These colors are part of the console’s color theme system, which can be edited or replaced.
